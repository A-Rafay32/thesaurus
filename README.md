## Features 
  * Shared Preferences 
  * Word of the day 
      * ( get notifications of random words )
      * ( used awesome_notifications package)
  * Redirects to web for words that are out of collection     
  * Provider for state management
  * Api Integration
  * Go-Router
  * MVVM Architecture
